# ZTheme - 现代化 Blogger 主题
ZTheme 是一款为 Google Blogger 平台打造的高性能、响应式、SEO 友好的现代化主题，兼顾美观与实用，适配移动端/桌面端，内置丰富的功能与无障碍设计，满足个人博客的全场景需求。

## ✨ 核心功能
### 1. 响应式设计
- 自适应移动端（768px/640px 断点）、平板、桌面端布局
- 移动端优化的导航、文章卡片、评论区样式

### 2. SEO 深度优化
- Open Graph/Twitter Card 社交标签自动生成
- robots 规则精细化（搜索页/归档页/404页 noindex）
- JSON-LD 结构化数据（BlogPosting 类型）
- RSS/ATOM/Sitemap 自动发现
- 自定义标题标签（首页/文章页/404页差异化）

### 3. 交互体验增强
- 阅读进度条（顶部实时进度）
- 图片灯箱（点击图片放大预览）
- 返回顶部按钮（滚动触发显示）
- 文章目录 TOC（可折叠，锚点联动）
- 代码块复制按钮（hover 显示，复制反馈）
- 评论区懒加载（解决国内加载慢问题）
- 加载更多按钮（替代分页，优化体验）
- 骨架屏（加载中占位，提升感知性能）
- 图片懒加载（按需加载，节省流量）

### 4. 无障碍与体验适配
- 键盘焦点样式（:focus-visible）
- 减少动效适配（prefers-reduced-motion）
- 打印样式优化（隐藏无关元素，适配纸张排版）
- 语义化 HTML 结构

### 5. 实用工具
- 文章阅读时间/字数统计
- 面包屑导航（首页→标签→文章）
- 相关文章推荐（基于标签）
- 内置搜索功能
- 版权声明区块（动态注入）
- PWA 支持（manifest.json 关联，添加到主屏幕）

### 6. 性能优化
- CDN 预连接/ DNS 预解析（常用域名）
- Bootstrap 5 轻量化引入
- 平滑滚动、过渡动画（性能优先）

## 🚀 安装方法
### 1. 下载主题文件
将本仓库的 `ZTheme.xml` 文件保存到本地。

### 2. 导入到 Blogger
1. 登录 Blogger 后台 → 主题 → 「备份/还原」
2. 点击「上传主题」，选择本地的 `ZTheme.xml` 文件
3. 点击「上传」，等待导入完成即可生效

### 3. 可选配置
- 自定义 `manifest.json`：在博客根目录上传 PWA 清单文件（需开启 Blogger 静态文件托管）
- 调整配色：修改 `b:skin` 区块内的 CSS 变量/颜色值
- 自定义导航：在 Blogger 后台「页面」中添加/修改导航页
